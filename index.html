Tarpo

Open source, cross platform Data Management desktop application for Dog Health programs in rural and remote Indigenous Australian communities, written entirely in Javascript.

Why

See blog.patspam.com/2008/tarpo

Install

Tarpo is a cross platform desktop application. Click the "Install Now" button below to install Tarpo onto your computer.

Please upgrade your Flash Player You do not appear to have Flash Player 6.0.65 or higher installed.
If you are installing for the first time, the Adobe AIR Runtime will be automatically installed.

Community

Visit the Tarpo Project Page on GitHub.

Visit the Issue Tracker to submit bugs and feature requests.

You can clone the project with Git by running:

$ git clone git://github.com/pdonelan/tarpo
