Install

Click the "Install Now" button below to Install Tarpo onto your computer.

Please upgrade your Flash Player This is the content that would be shown if the user does not have Flash Player 6.0.65 or higher installed.
If you are installing for the first time, the Adobe AIR Runtime will be automatically installed (or you can download it manually here).
