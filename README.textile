h1. Introduction

"Tarpo":http://pdonelan.github.com/tarpo is an open source, cross platform Data Management desktop application for Dog Health programs in rural and remote Indigenous Australian communities, written entirely in Javascript. 

Visit the "Download Page":http://pdonelan.github.com/tarpo to get started.

Visit the "Issue Tracker":http://github.com/pdonelan/tarpo/issues to report bugs and request new features.

Visit the "Project Page on GitHub":http://github.com/pdonelan/tarpo to get involved in development.
